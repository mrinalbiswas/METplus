359  to be looped over based on user settings, the values
360  in the list, and process being run.

362  Args:

368  configuration information

370  @returns tuple containing lists_to_group_items ( list of

373  the list names whose items are being looped over)

381  for missing_config in missing_config_list:
382

394  c_dict['GROUP_LIST_ITEMS'].append(missing_config)
395 
396  # otherwise add to loop list

402  c_dict['GROUP_LIST_ITEMS'].append(missing_config)
